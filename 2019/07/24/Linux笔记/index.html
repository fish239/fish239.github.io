VLAN接口

VLAN(Virtual LAN)，翻译成中文是“虚拟局域网”.网络接口名称如:eth0.1、eth0.2

添加VLAN接口：
vconfig add eth0 100
ifconfig eth0.100 2.2.2.2 netmask 255.255.255.0 down
PS:如果没有vconfig需要yum安装：yum install vconfig
